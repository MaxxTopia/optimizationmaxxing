faq

Frequently asked

Will this brick my PC?

No. Every tweak is undo-safe — we record the prior value before changing anything, in a SQLite snapshot store. One-click revert per tweak. We refuse to write BIOS in software (that's where bricking happens). For BIOS, we tell you what to type and you flip it manually.

Is this safe to use with Vanguard / EAC / BattlEye?

Most tweaks are anti-cheat-safe (registry / scheduler tweaks). A few are flagged with anti-cheat-risk badges in the catalog. We don't ship a kernel driver. Read the Toolkit research card on AMD/Intel + Vanguard caveats before applying high-risk tweaks.

Why is the installer unsigned?

EV code-signing certs cost $300+/yr. We're getting one before public launch; for the pilot, SmartScreen warns. Click "More info" → "Run anyway." SHA-256 is published so you can verify nothing was tampered with.

What's "spec-aware curation"?

Each tweak in our catalog has a "requires" predicate: e.g. "Intel 12th gen+ AND Win11 22H2+ AND not laptop." If your rig doesn't match, the tweak is hidden or shown grayed-out with the gating reason. Beats one-size-fits-all that ships you Hyper-V tweaks on a Mac VM.

Why is Paragon $20 and you're $8?

Paragon's moat is pro-player endorsements and a manual 1-on-1 service tier — they charge for relationship + brand. We're charging for the software + content. Different value prop. We're priced to be a no-brainer for solo gamers.

Does it work on AMD or only NVIDIA?

Both. Spec detection identifies your GPU vendor (NVIDIA/AMD/Intel Arc) and tweaks gate accordingly. NVIDIA Profile Inspector integration is NVIDIA-only by design (Phase 4d).

Where's the source?

Public GitHub repo coming with the v0.2 launch. For the pilot, hash + Discord support is the trust path.
